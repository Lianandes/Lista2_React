# Lista 2 - React

Projeto desenvolvido para praticar e aprofundar os conceitos de React através da resolução dos exercícios propostos na Lista 2.

## 🚀 Objetivo

O objetivo deste projeto é fortalecer os conhecimentos em React por meio da implementação prática dos exercícios, desenvolvendo aplicações mais organizadas e reutilizáveis.

## 🛠 Tecnologias utilizadas

- React
- JavaScript
- HTML5
- CSS3
- Node.js
- Vite

## ✨ Funcionalidades

- Desenvolvimento dos exercícios da Lista 2
- Componentização
- Organização da aplicação
- Estruturação de interfaces
- Prática de conceitos do React

## 📚 Conceitos praticados

- Componentes
- JSX
- Props
- Estrutura de projetos
- Reutilização de código
- Organização de interfaces

## 📌 Observação

Este projeto foi desenvolvido com fins educacionais para aprimorar as habilidades em React através da resolução de exercícios práticos.
